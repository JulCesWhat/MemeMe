# MemeMe

iOS app writen in swift.

## Requirements

Swift, xCode, and macOS

## Set up

Clone repo by `git clone https://github.com/JulCesWhat/MemeMe.git` and open it with xCode.

## Build

Press the play button in the left top corner.
